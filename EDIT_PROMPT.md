# VP Scene Edit Instructions

## 📌 Tag ฉากใหม่ (แนบภาพ)

```
tag ฉากนี้แล้ว apply เข้า VP Library ให้เลย ID = VP-0XX
```

---

## ✏️ แก้ข้อมูลฉาก

```
แก้ฉาก VP-0XX:
- world: interior-office
- energy: balanced
- title: Modern Loft Interior
- title_th: ล็อฟท์สมัยใหม่
- context_tags: [talk-interview, news-report]
- notes: (ถ้าต้องการเปลี่ยน)
```

---

## 🗂 Field ทั้งหมดที่แก้ได้

| field | ค่าที่ใช้ได้ |
|---|---|
| world | urban-city / interior-office / nature-outdoor / industrial / scifi-future / stylized-abstract / heritage-culture |
| energy | intense / balanced / open |
| context_tags | news-report / talk-interview / analysis-data / field-onsite / opener-drama / social-short / event-special |
| color_temp | cool / warm / neutral / mixed |
| brightness | dark / low / medium / bright / high-key |
| has_particles | true / false |
| title | ชื่อ EN |
| title_th | ชื่อ TH |
| notes | production notes ภาษาไทย |
| status | active / draft / archived |

---

## 📋 ดูข้อมูลฉากปัจจุบัน

```
ดูข้อมูลทุกฉากใน VP Library ให้หน่อย
```

```
ดูข้อมูลฉาก VP-0XX
```
