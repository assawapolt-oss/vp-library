🤖 Local AI Tag (Ollama)

ใช้ Ollama รัน vision model บนเครื่องของคุณเอง — ไม่ต้องใช้ API Key และข้อมูลไม่ออกอินเทอร์เน็ต

กด ดึงรายการ เพื่อโหลด model ที่ติดตั้งแล้ว หรือพิมพ์ชื่อเองได้เลย
📋 วิธีติดตั้ง Ollama
1. ดาวน์โหลดจาก ollama.com
2. เปิด Terminal แล้วรัน: ollama pull moondream
3. รัน: ollama serve (ถ้ายังไม่ได้เปิด)
หรือใช้ Python script สำหรับ batch tag ทุก scene พร้อมกัน:
cd VP_Library && python3 vp-ai-tag.py
